UP WH LOC FINDER - Offline Version
====================================

How to use:
1. Unzip the folder.
2. Open 'index.html' in your web browser.
   (You can double-click it or drag it into a browser window.)
3. Once open, type or paste a MATERIAL_ID or BARCODE_NUMBER in the search box.
4. The system will instantly display all matching records.

Files included:
- index.html (main app)
- app.js (logic)
- styles.css (design)
- warehouse_data.json (converted data from your Excel)
- README.txt (this file)

No internet or server is required. Everything runs locally in your browser.
